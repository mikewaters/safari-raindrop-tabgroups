## Tools & Libraries

### Node.js Packages

Common packages used in Raycast extensions:

- **@raycast/api**: Core Raycast API
- **@raycast/utils**: Utility functions and hooks
- **node-fetch**: HTTP requests (if not using useFetch)
- **rss-parser**: Parse RSS feeds
- **cheerio**: HTML parsing
- **date-fns**: Date manipulation
- **lodash**: Utility functions
- **axios**: Alternative HTTP client

### Development Tools

- **Visual Studio Code**: Recommended IDE
  - Raycast VSCode extension available
  - TypeScript support
  - Debugging support

- **React Developer Tools**: For debugging UI
- **Postman/Insomnia**: For API testing

## Learning Path

### For Beginners

1. Read **Getting Started Guide**
2. Follow **Create Your First Extension** tutorial
3. Study **Hacker News Example**
4. Explore **UI Components** documentation
5. Build a simple extension (todo list, bookmarks)

### For Intermediate Developers

1. Study **React Hooks** documentation
2. Learn **Storage** and **Preferences** APIs
3. Implement **API integration** with authentication
4. Add **Navigation** between screens
5. Study real extensions on GitHub

### For Advanced Developers

1. Optimize performance with caching
2. Implement complex state management
3. Build extensions with multiple commands
4. Contribute to open-source extensions
5. Publish to Raycast Store

### Key Commands

```bash
# Create extension
# Use Raycast: "Create Extension" command

# Install dependencies
npm install

# Start development
npm run dev

# Build extension
npm run build

# Lint code
npm run lint

# Fix linting issues
npm run fix-lint
```

### Import Cheat Sheet

```typescript
// UI Components
import { List, Grid, Detail, Form } from "@raycast/api";

// Actions
import { ActionPanel, Action } from "@raycast/api";

// Feedback
import { showToast, Toast, confirmAlert, Alert, showHUD } from "@raycast/api";

// Utilities
import { useNavigation, getPreferenceValues, Clipboard } from "@raycast/api";

// Storage
import { LocalStorage, Cache } from "@raycast/api";

// Hooks
import { usePromise, useFetch, useCachedState, useForm } from "@raycast/utils";

// Icons & Colors
import { Icon, Color } from "@raycast/api";

// Environment
import { environment, open, showInFinder } from "@raycast/api";
```
